Questa cartella contiene i file per leggere e analizzare un tipo di file CSV in cui vengono specificati i consumi di tot. strumenti presenti in azienda. 

Il file main permette di creare una window tramite tkinter per la scelta del file da analizzare e comunica con reader per leggere i dati del file e modificarli
per ritornare come risultato un dataframe di valori (es. consumo totale degli strumenti, consumo maggiore in percentuale ecc). 

Questi risultati vengono visualizzati in stringa su una textbox il cui contenuto vinee rigenerato ogni qualvolta che un file viene scelto.
